willow-tree.js

The willow-tree.js plugin is implemented similar to many other tree libraries made for jQuery, except, that it does not initialize any additional markup itself. Instead it assumes that you provide the necessary markup yourself, so that we are able to bind long-living event handlers to a given context.

With that context event handlers in place, we are able to inject new trees to that context, without the necessity to reinitialize the tree library again.
